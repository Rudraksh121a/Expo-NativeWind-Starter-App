# Expo-NativeWind-Starter-App

A React Native starter app built with Expo and NativeWind for rapid mobile development.

## Prerequisites

Before you begin, ensure you have the following installed:

- Node.js (version 18 or later)
- npm or yarn package manager

## Installation

### 1. Install Expo CLI globally

```bash
npm install -g @expo/cli
```

### 2. Clone and setup the project

```bash
# Clone the repository
git clone <your-repo-url>
cd Expo-NativeWind-Starter-App


# Install dependencies
npm install

# Start the development server
npx expo start
```

## Development

To start developing:

1. Run `npx expo start` to start the Metro bundler
2. Use the Expo Go app on your mobile device to scan the QR code
3. Or press `i` for iOS simulator / `a` for Android emulator
